金银的避险主要还是体现在抗通胀，避免金融市场体系崩溃。虽然宽松了，但是货币要通过经济活动才能推高通胀。这次美股下跌与2008年危机并不相似，这次主要还是实体经济出现了停滞。以上2个原因是我认为金银不会继续大幅上涨的原因。
